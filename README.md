# stronghold-sultan

The Sultan sound pack for [peon-ping](https://github.com/PeonPing/peon-ping) — voice lines from The Sultan AI lord in Stronghold Crusader.

## Install

```bash
peon packs install stronghold-sultan
```

## Adding sounds

Drop your `.mp3`/`.wav`/`.ogg` files into `sounds/` and update `openpeon.json` to reference them.

Expected files per category:

| File | Category |
|---|---|
| `start_1.mp3`, `start_2.mp3` | Session start (greeting) |
| `complete_1.mp3`, `complete_2.mp3` | Task complete |
| `permission_1.mp3` | Input required |
| `error_1.mp3` | Task error |
| `limit_1.mp3` | Resource limit |
| `spam_1.mp3` | User spam |
